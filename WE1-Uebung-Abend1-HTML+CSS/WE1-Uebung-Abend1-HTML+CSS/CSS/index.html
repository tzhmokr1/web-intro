Quiz 2

Folgendes CSS und HTML ist gegeben 

      article > h1 {
        background-color: orange;
      }
      p > p {  
        background-color: orange;
      }    
    
    
      <h2>Question 2</h2>
      <article>  
        <header>
          <h1>A Title</h1>
        </header>
        <section>
          <p>start of a section       
            <p> a paragraph in paragraph</p>    
          </p>  
        </section>
      </article>

Wie Sie hier sehen haben beide Regeln keinen Effekt.

Erklären Sie warum weder die eine noch die andere Regel einen Effekt hat

Regel 1 hat keinen Effekt, weil es kein h1 gibt welches als DIREKTES Kind in einem article enhalten ist

Regel 2 hat keinen Effekt, weil ein p-Elemente in einem p-Element nicht erlaubt ist. Im geparsten DOM liegt das zweite p neben dem anderen p (Geschwister Elemente). Entsrechend gibt es im DOM kein p welches in einem anderen p liegt.

Quiz 3

Folgendes HTML und CSS Stylesheet ist gegeben. 
      
        <div id=style-box class="style-box">
        </div>      
      
    

      #style-box::before {
        content: "A";
      }
      .style-box::before {
        content: "B";
      }

FRAGE: Was erscheint im div: "A" oder "B" (d.h. welche der beiden Regeln ist stärker?)

Es erscheint A. Die erste Regel ist stärker, da die Regeln mit einem ID-Selektor stärker ist als die Regel einem Class-Selektor

Quiz 4

Gleiches HTML wie in Aufgabe 3 ist gegeben mit folgendem CSS Stylesheet. 

      * .style-box::before {
        content: "A";
      }
  
      .style-box::before {
        content: "B";
      }

FRAGE: Was erscheint im div: "A" oder "B" (d.h. welche der beiden Regeln ist stärker?)

Es erscheint B. Beide Regeln sind gleich stark, da der universelle Selektor (*) keinen Einfluss hat. Wenn Regeln gleich stark sind, dann gewinnt die letzte.

Quiz 5

Gleiches HTML wie in Aufgabe 3/4 ist gegeben mit folgendem CSS Stylesheet. 

      :root .style-box::before {
        content: "A";
      }
  
      body .style-box::before {
        content: "B";
      }

FRAGE: Was erscheint im div: "A" oder "B" (d.h. welche der beiden Regeln ist stärker?)

Es erscheint A. (Pseudo)-Class-Selektoren sind stärker als Element-Selektoren.

Quiz 6

Gleiches HTML wie in Aufgabe 3/4 ist gegeben mit folgendem CSS Stylesheet. 

      :root .style-box.style-box::before {
        content: "A";
      }
      :root .style-box::before {
        content: "B";
      }

FRAGE: Was erscheint im div: "A" oder "B" (d.h. welche der beiden Regeln ist stärker?)

Es erscheint A. Die Wiederholung des Klassen-Selektors bewirkt, dass der Selektor der ersten Regel eine höhere "Specificity" hat.
