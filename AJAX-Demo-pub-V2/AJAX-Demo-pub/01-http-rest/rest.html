  • Use Insomnia or Postman to create a POST request and create a new song entry. Make sure header is set to 
    'Accept': 'application/json',
'Content-Type': 'application/json'
    and a json body is set e.g. 
    {
    "title": "My New Song",
    "artist": "TBD",
    "rating": 0
}
    or use this curl request (unix shell) 
    curl --request POST \
  --url http://localhost:3000/songs/ \
  --header 'accept: application/json' \
  --header 'content-type: application/json' \
  --data '{"title":"My New Song 2","artist":"TBD","rating":0}'
    or use this fetch request (JS console) 
    fetch("http://localhost:3000/songs/", {
  "method": "POST",
  "headers": {
    "accept": "application/json",
    "content-type": "application/json"
  },
  "body": JSON.stringify({
    "title": "My New Song 3",
    "artist": "TBD",
    "rating": 0
  })
})
.then(response => {
  console.log(response);
})
.catch(err => {
  console.error(err);
});
